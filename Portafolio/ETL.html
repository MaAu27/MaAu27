Extracción de datos en la web

-Extraer

Identificación dentro de la estructura HTML de la página web las variables a extraer.

-Transformar

Manipulación de las variables extraidas para procesarlas en datos con valor para el negocio.

-Cargar

Depositar los datos estructurados en el formato establecido en el sistema de destino.

-Software

Python, Jupyter Notebook y Anaconda para la extracción y manipulación de datos.
